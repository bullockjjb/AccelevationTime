# AccelevationTime

Prototype of the "Acceleration Time" Phaser game. Open `index.html` in a modern browser. The canvas now resizes to your browser window.

Press **any key** (or click) on the main menu to start. The menu shows the Accelevation logo and features the player performing a couple jumps with a sound effect before running across the screen. All images and audio used are embedded as Base64 strings in the JavaScript.
