# AccelevationTime

Prototype for the "Acceleration Time" Phaser game. Press **any key** on the
main menu to begin playing.

All graphics are embedded directly in the JavaScript as base64 strings to keep
the repository text only. The player sprite now comes from `Maincharacter.png`
provided by the user. The main menu also displays the Accelevation logo from
the company website. A new "Rogue AI" enemy uses artwork loaded from an online
image. Open `index.html` in a modern browser to play the demo level.
