# AccelevationTime

Prototype for the "Acceleration Time" Phaser game. Press the **Spacebar** on
the main menu to begin playing.

All graphics are embedded directly in the JavaScript as base64 strings to keep
the repository text only. The player sprite now comes from `Maincharacter.png`
provided by the user. The main menu also displays the Accelevation logo from
the company website. Open `index.html` in a modern browser to play the demo
level.
